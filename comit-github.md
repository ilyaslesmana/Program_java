git add .Staging: Menyiapkan semua draf laporan masuk ke meja manajer untuk disetujui.
git commit -m "pesan" Approval: Memberikan stempel resmi pada dokumen beserta catatan perubahan (timestamp).
git push origin main Deployment: Mengirimkan dokumen resmi tersebut ke server pusat agar bisa diakses oleh stakeholder lain.
